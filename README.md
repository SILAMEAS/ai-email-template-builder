## Getting Started

First, run the development server:

Frontend 

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Second, run the development server:

Backend

```bash
npm run dev-convex
# or
yarn dev-convex
# or
pnpm dev-convex
# or
bun dev-convex
```
